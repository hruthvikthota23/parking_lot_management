# 🅿️ Parking Lot Management System

A console-based Parking Lot Management System developed using Java and Object-Oriented Programming (OOP) concepts. The project manages vehicles, parking slots, and parking operations efficiently.

## 🚀 Features
- Add and manage parking slots
- Park and remove vehicles
- Check available parking spaces
- Track occupied and empty slots
- Manage different vehicle types

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework

## 🧩 OOP Concepts Applied
- Encapsulation
- Inheritance
- Abstraction
- Polymorphism
- Association
- Aggregation

## 📂 Classes
- Vehicle
- Car
- Bike
- ParkingSlot
- ParkingLot
- ParkingRecord
- Main

## 📌 Project Overview
The system follows a modular object-oriented design where ParkingLot manages slots, vehicles, and parking records. Vehicle classes represent different vehicle types, while parking records maintain entry and exit details.
